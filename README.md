# Activity-Recognition

## 项目内容

- 模型训练及实验分析

- 手机行为识别APP开发

## 所需工具

实验实例全部运行于Python 3.6环境，尚需引入以下扩展包。

- sklearn

- pandas

- numpy

- matplotlib

手机APP开发需如下支持。

- JDK 9

## 实验数据

来自[WISDM](./data_set) 数据集，志愿者数为36 人，共1098207 个采样点，包含以下六种行为。

- 静坐(Sitting)

- 站立(Standing)

- 上楼(Upstairs)

- 下楼(Downstairs)

- 步行(Walking)

- 慢跑(Jogging)
